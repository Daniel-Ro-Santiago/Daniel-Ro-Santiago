## Hi there

I'm Daniel, an actuarial science student based in Mexico.

I'm interested in applying statistics, mathematics, and data science to develop better solutions for complex problems.

I develop projects in areas such as actuarial modeling, machine learning, financial analysis, computational biology, and software development.
